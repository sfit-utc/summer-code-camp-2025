## Thực hành Docker cơ bản
### Docker CLI
1. Kiểm tra phiên bản Docker:
   ```bash
   docker --version
   ```
2. Kiểm tra trạng thái Docker:
   ```bash
   docker info
   ```
3. Xem documentation của Docker:
   ```bash
   docker --help
   ```
